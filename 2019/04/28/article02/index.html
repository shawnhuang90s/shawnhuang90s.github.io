编辑器的选择

  1. Hexo下所写博文的文件类型都是 Markdown 文件
  2. 在写博文之前，我们先要做的一件事就是选择一款 Markdown编辑器
  3. 如果你只是写字、写代码和上传图片比较多，对表格之类的没什么要求，强烈建议使用 Typora
  4. 这款编辑器的优点很多，比如几乎不占内存，使用特别方便，还有最重要的一点：所看即所得！
  5. Markdown 具体语法介绍请移步：献给写作者的 Markdown 新手指南

  1. 因为七牛云需要备案，否则图片链接会失效，所以如果你的网站没有备案或者不想备案的话，考虑换个图床
