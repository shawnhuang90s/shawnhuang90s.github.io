Python2 和 Python3 的区别

  1. Python2 的默认编码方式是 ASCII

    Python3 的默认编码方式是 Unicode

  2. Python2 中默认都是经典类，只有显示继承 Object 类才是新式类(Python2.2 版本开始引入)

    Python3 中默认都是新式类，不必显示继承 Object 类

  3. Python2 中 print 是一条语句

    Python3 中 print 是一个函数，因此后面的内容要用括号括起来

  4. Python3 中这些方法不再返回 list 对象，但是可以通过 list 函数强行转换：

  5. Python2 中的 range 返回的是一个列表

    Python3 中的 range 返回的则是一个迭代器

编程语言的分类

  1. 解释型语言，特点：
    • 逐行解释，逐行执行
    • 代表语言：Python \ PHP
    • 优点：开发效率高，跨平台
    • 缺点：执行效率较低
  2. 编译型语言，特点：
    • 将程序全部编译成二进制，然后再执行
    • 代表语言：C \ Object-C
    • 优点：执行效率高
    • 缺点：开发效率低，不能跨平台

变量命名规则

  1. 变量名必须由数字、字母或者下划线的任意组合
  2. 变量名不能是数字开头
  3. 不能使用 Python 中的关键字
  4. 变量名要具有可描述性
  5. 变量名不能过长
  6. 变量名不能用中文，尽量不要用拼音
  7. 变量名用下划线命名方式更好

字符编码

  1. 编码的定义——人类能识别的文字与机器能够读懂的二进制的对应关系表

  2. 字符编码的分类：

    • ASCII —— 最早的编码本，只包含：数字、字母和特殊字符，8 位(bit) 一个字节表示一个字符
    • Unicode —— 万国码，32 位四个字节表示一个字符
    • UTF-8 —— 英文：一个字节表示一个字符；欧：两个字节表示一个字符；中：三个字节表示一个字符
    • GBK —— 英文：一个字节表示一个字符；中：两个字节表示一个字符

  3. 注意：数字、字母和特殊字符映射的是 ASCII
